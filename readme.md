# YelpCamp

Yelpcamp is a Node.js web app that is used to upload and rate popular campsites.

## Installation

1. Clone this repository
2. run `npm install` or `yarn install` to get all of the dependencies
3. create a `.env` file in the root of the repo
4. inside the `.env` file, add a line: `PASSPORT_SECRET_KEY = 'YOUR SECRET KEY HERE'`
5. download and install mongodb

## Running this app

1. run `node app.js`, it will start a local server on `port 3000`
2. go to `localhost:3000` in your browser.
